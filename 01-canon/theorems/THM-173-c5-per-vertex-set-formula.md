# THM-173: Per-Vertex-Set Formula for c5_dir

**Status:** VERIFIED (n=5 exhaustive, n=6 500/500, n=7 200/200)
**Proved by:** kind-pasteur-2026-03-13-S61
**Depends on:** THM-172 (c5 is lambda-determined)

## Statement

For any tournament T on n vertices, the total directed 5-cycle count is:

$$c_{5,\text{dir}}(T) = \sum_{V \in \binom{[n]}{5}} f(\text{hist}(\Lambda_V))$$

where $\Lambda_V$ is the restricted lambda graph on 5-vertex set V (using only witnesses within V), hist is the histogram $(n_0, n_1, n_2, n_3)$ counting pairs with each lambda value, and f is the lookup table:

| n0 | n1 | n2 | n3 | c5 | Description |
|----|----|----|----|----|-------------|
| 10 |  0 |  0 |  0 |  0 | Transitive |
|  7 |  3 |  0 |  0 |  0 | 1 three-cycle |
|  5 |  4 |  1 |  0 |  0 | 2 three-cycles |
|  3 |  5 |  2 |  0 |  1 | 3 three-cycles (star) |
|  3 |  6 |  0 |  1 |  1 | 3 three-cycles (hub pair) |
|  1 |  6 |  3 |  0 |  2 | 4 three-cycles (star-like) |
|  0 |  9 |  0 |  1 |  3 | 4 three-cycles (hub pair) |
|  2 |  4 |  4 |  0 |  1 | 4 three-cycles (distributed) |
|  0 |  5 |  5 |  0 |  2 | 5 three-cycles (regular) |

## Key Identities

1. **Restricted lambda decomposition:** For V = [n]\{k}, $\Lambda_V(u,v) = \lambda(u,v) - w(k,u,v)$ where $w(k,u,v) = 1$ iff k witnesses the 3-cycle containing {u,v}.

2. **Lambda determines c5:** The lambda histogram on any 5-vertex tournament completely determines the directed 5-cycle count. This is because at n=5, there are only 9 distinct lambda histograms (corresponding to the 9 distinct "lambda multisets").

3. **Per-vertex-set summation:** Since each restricted lambda depends only on the full lambda graph and the vertex set V, the total c5 is a function of the full lambda graph.

## Proof Sketch

1. At n=5, exhaustive enumeration shows exactly 9 lambda histogram classes, each with a unique c5 value.
2. For n > 5, $c_{5,\text{dir}}(T) = \sum_{V \in \binom{[n]}{5}} c_5(T[V])$ where $c_5(T[V])$ is the directed 5-cycle count on the 5-vertex induced sub-tournament.
3. Each $c_5(T[V])$ depends only on the lambda histogram of $T[V]$, which is determined by the full lambda graph of T.

## Corollary: dc5 = 0 under Vitali atoms

Since Vitali atoms preserve the full lambda graph, and c5_dir is a function of the lambda graph (via this per-vertex-set formula), dc5 = 0 follows trivially.

## Connection to the sigma-algebra hierarchy

The invariant $c_{5,\text{dir}}$ is "lambda-measurable" — it factors through the projection T → L(T). In contrast, $c_{7,\text{dir}}$ is NOT lambda-measurable (5 ambiguous lambda groups at n=7) but IS measurable with respect to the finer sigma-algebra generated by (lambda, sigma), where sigma(u,v) = #{common successors} + #{common predecessors} of u and v.
